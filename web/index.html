QuuBee — PC-98 フリーソフトのブラウザプレイヤー

フリーソフトの書庫 (.lzh / .zip) や ディスクイメージ (.d88 / .fdi / .hdm 等) を、
このパネルかゲーム画面にドロップしてください。
（[Open] でファイル選択も可）

  ・ファイル一覧から readme などを開いて読みながら、
  ・EXE / COM を選んで [▶ Run] で起動できます。
  ・ドロップはいつでも「新しく開く」。重ねるときは [＋Add]
  　（HD インストール風・パッチ当て・[Save] したセーブや自作データの
  　　単体読み戻しに。単体ファイルは半角英数 8.3 名のみ）。
  ・ディスクイメージはブートせず、中のファイルだけを取り出します
  　（自己起動・非 FAT のイメージは対象外）。
  ・左右の境目にある ▸ でゲーム画面を最大化（◂ で戻る）。

ゲームは同梱していません。お手元のフリーソフトをどうぞ。
画面クリックでマウスキャプチャ（ESC で解除）。

QuuBee — a browser player for PC-98 freeware

Drop a freeware archive (.lzh / .zip) or a disk image
(.d88 / .fdi / .hdm etc.) onto this panel or the game screen.
(You can also pick a file with [Open].)

  - Open the readme and other files from the list, then
  - select an EXE / COM and press [▶ Run] to start it.
  - A drop always opens fresh. Use [＋Add] to layer files on top
    (HD-install style, patches, or re-importing saves and your own
    data exported with [Save]. Single files: ASCII 8.3 names only).
  - Disk images are never booted — only the files inside are
    extracted (self-booting / non-FAT images are not supported).
  - The ▸ handle on the seam maximizes the game screen (◂ restores).

No games are bundled. Bring your own freeware.
Click the screen to capture the mouse (press ESC to release).
