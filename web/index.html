QuuBee — PC-98 フリーソフトのブラウザプレイヤー

フリーソフトの書庫 (.lzh / .zip) や ディスクイメージ (.d88 / .fdi / .hdm 等) を、
このパネルかゲーム画面にドロップしてください。
（[Open] でファイル選択も可）

  ・ファイル一覧から readme などを開いて読みながら、
  ・EXE / COM を選んで [▶ Run] で起動できます。
  ・画像 (.MAG / .PI) や音楽 (.M) は単体でも開けます
  　（ドロップ / [Open] / [＋Add]。見る・聴くだけなので前のゲームは閉じません）。
  ・ドロップはいつでも「新しく開く」。重ねるときは [＋Add]
  　（HD インストール風・パッチ当て・[Save] したセーブや自作データの
  　　単体読み戻しに。セーブ等の単体ファイルは半角英数 8.3 名のみ）。
  ・ディスクイメージはブートせず、中のファイルだけを取り出します
  　（自己起動・非 FAT のイメージは対象外）。
  ・左右の境目にある ▸ でゲーム画面を最大化（◂ で戻る）。

ゲームは同梱していません。お手元のフリーソフトをどうぞ。
画面クリックでマウスキャプチャ（ESC で解除）。

────────────────────────────────────────

QuuBee — a browser player for PC-98 freeware

Drop a freeware archive (.lzh / .zip) or a disk image
(.d88 / .fdi / .hdm etc.) onto this panel or the game screen.
(You can also pick a file with [Open].)

  - Open the readme and other files from the list, then
  - select an EXE / COM and press [▶ Run] to start it.
  - Image (.MAG / .PI) and music (.M) files open on their own too
    (drop / [Open] / [＋Add] — just to view or play; your game stays loaded).
  - A drop always opens fresh. Use [＋Add] to layer files on top
    (HD-install style, patches, or re-importing saves and your own
    data exported with [Save]. Single saved files: ASCII 8.3 names only).
  - Disk images are never booted — only the files inside are
    extracted (self-booting / non-FAT images are not supported).
  - The ▸ handle on the seam maximizes the game screen (◂ restores).

No games are bundled. Bring your own freeware.
Click the screen to capture the mouse (press ESC to release).

「QuuBee」は、PC-98のフリーソフト・同人ソフト文化を継承するための、
ブラウザで動くプレイヤーです。

対象は、かつて開発者自身の手で配布されたソフトやゲーム。
同梱のマニュアルや、作者の心情が垣間見えるテキストまで含めた、
「PC-98フリーソフト文化」の再体験を目指しました。

市販ソフトの動作は目的ではありません。ディスクイメージから
ブートする機構も、意図的に持たせませんでした。つまり、
ブートを前提とする市販ソフトは原理的に動きません。

DOSの振る舞いを実際のDOSを使わずに再実装し、BIOSも実際の
BIOSの動作を模倣する仕組みを構築しました。画面のフォントは
NECのROMから抜き出したものではなく、フリーフォント由来の
代替ビットマップです。ハードウェアの再現は、オープンソースの
エンジン（NP2kai）が担っています。つまり、このプレイヤーは、
フリーなライセンスに基づくものだけで構成されています。
そこで動くのは、あなたの手元にあるソフト、それだけです。

「QuuBee」はエミュレータではありません。
PC-98文化を継承し、再発見するためのツールです。

────────────────────────────────────────

QuuBee is a player that runs in your browser, made to carry
on the culture of PC-98 freeware and doujin software.

It is for the software and games once distributed by the
developers' own hands. The aim is to relive the PC-98
freeware culture as a whole — including the bundled manuals
and the texts in which the authors' personalities still
shine through.

Running commercial software is not a goal. We also chose not
to build any mechanism for booting disk images. In other
words, commercial titles that require booting simply cannot
run here.

DOS behavior is re-implemented without using any real DOS,
and the BIOS is built to imitate how the real one behaves.
The on-screen font is not extracted from NEC's ROM but is a
substitute bitmap derived from free fonts. The hardware is
reproduced by NP2kai, an open-source engine. In other words,
this player is built entirely from freely licensed
components. What runs on it is the software in your own
hands — that, and nothing else.

QuuBee is not an emulator.
It is a tool for inheriting and rediscovering the PC-98
culture.
