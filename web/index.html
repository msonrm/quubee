QuuBee — PC-98 フリーソフトのブラウザプレイヤー

フリーソフトの書庫 (.lzh / .zip) を、右のパネルかゲーム画面にドロップしてください。
（「書庫をドロップ」をクリックでファイル選択も可）

  ・ファイル一覧から readme などを開いて読みながら、
  ・EXE / COM を選んで [▶ Run] で起動できます。
  ・「＋追加」で別の書庫を同じ場所に重ねて展開（HD インストール風）。
  ・右上「▦ パネル」でこの欄を隠すと画面が広がります。

ゲームは同梱していません。お手元のフリーソフトをどうぞ。
画面クリックでマウスキャプチャ（ESC で解除）。
